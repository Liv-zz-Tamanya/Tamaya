Part 3 — API 명세 (25+ endpoints)

Base URL: https://api.tamaya.app/v1 (prod 미정, dev http://localhost:8000) 인증: Authorization: Bearer <jwt> (익명 토큰 포함) Content-Type: application/json · 시간: ISO 8601 UTC · 에러: { "detail": "한국어 메시지" } (FastAPI 표준)

3.1 인증 (3)

POST /v1/auth/kakao
  body: { code: string, device_id: uuid }
  resp 200: {
    access_token: jwt(5min),
    refresh_token: jwt(14d),
    user: { id, kind:"kakao", name, needs_onboarding: bool },
    onboarding_step: 0..5
  }
  errors: 400 invalid_code · 401 kakao_revoked

POST /v1/auth/anonymous
  body: { device_id: uuid }   # 클라이언트가 localStorage에 발급·저장
  resp 200: { access_token, refresh_token, user:{kind:"anonymous"}, onboarding_step }
